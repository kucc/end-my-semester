## 시작하기

```shell
git clone https://github.com/kucc/end-my-semester.git && cd end-my-semester && cp .env.example .env.local
```

이후 `.env.local` 파일에서 환경 변수를 설정해주세요.

```shell
npm install && npm run dev
```
